6. DATA AND PRIVACY

PC Brain is designed as a locally-governed system. The following data practices apply:

6.1 Data That Stays Local

All of the following are stored exclusively on your machine and are never transmitted to PC BRAIN LLC servers:

Your goals, tasks, memories, chat history, and action outcomes (brain.db).

Your API keys and configuration (.env file).

Device credentials, file events, session handoffs, and agent logs.

Screen captures, chat attachments, and research files.

6.2 Data Transmitted to Third-Party LLM Providers

When you use AI features, your prompts and contextual system data (active window title, CPU/RAM stats, your configured goals and tasks) are sent to the LLM provider YOU have configured (e.g., OpenAI, Anthropic, Google). This data is sent to YOUR account at those providers — not to PC BRAIN LLC servers. You are subject to those providers' privacy policies.

6.3 Data Transmitted to PC BRAIN LLC

The Software contacts PC BRAIN LLC servers only for:

License key activation (license key + machine fingerprint hash).

Update checks every 4 hours (User-Agent header only: "PCBrain-Updater" — no personal data).

6.4 Optional Telemetry

Telemetry is OFF by default. If you explicitly enable it, the Software may send: software version, node role, platform type, and truncated error messages (max 200 characters). No user content, file paths, or personal identifiers are included.

7. AUTOMATIC UPDATE CHECKS

The Software automatically checks api.pcbrain.ai/updates every 4 hours for available updates. No personal data is transmitted during this check. Software updates are never installed automatically — all updates require your explicit action.
