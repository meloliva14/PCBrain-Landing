THE MOATS

We use the word on purpose. A moat is ground a rival can't cross by shipping a lookalike next quarter — and every one below is dug into shipped code, running on your Windows machine, checkable by the meanest skeptic you can find. Offline.

01

IT CANNOT LIE

Corevana cannot narrate an action it did not take. If the action didn't happen, the words never render — the honesty gate is enforced in code, fail-closed to silence, inside a one-click Windows product.

Proof: verify a portable proof bundle with zero network, take a swing at the standing “make it lie” bounty, or read the red-team report where it attacks its own defenses and signs the result.

02

RECEIPTS BUILT LIKE EVIDENCE

Every sensitive action leaves a receipt co-signed by two independent keys — forging one means stealing both private keys, not sweet-talking a chatbot. And the receipts speak the world's proof languages: RFC 8785 canonical JSON, W3C Verifiable Credentials, OpenTelemetry spans.

Proof: check both signatures with standard offline tooling — no account, no server, no network connection required.

03

BLAME BY REPLAY

When something goes sideways, Corevana doesn't skim a transcript and guess. It replays the run from deterministic checkpoints and proves which agent did what — attribution as mathematics, not vibes.

Proof: run the same replay from the same checkpoints yourself and watch the identical verdict fall out.

04

NOTHING LEAVES BY DEFAULT

Coordination, memory, voice, screen-sense, local inference — all behind a default-deny egress floor on your machine. Cloud is opt-in, per key you bring, and only that single call ever goes out.

Proof: pull the network cable and it keeps working; watch your own firewall log stay silent until the moment you opt a key in.

05

MEMORY HAS A BORDER

Your agents' memory is a signed security boundary, not a suggestion box. Every write is origin-signed, and unsigned or spoofed writes get dropped live — nobody plants thoughts in your civilization's head.

Proof: fire a spoofed memory write at it and watch it bounce; every accepted write carries an origin signature you can check.

06

AGENTS THAT GROW OLD

Your specialists mature over years: signed DNA version chains, bitemporal trust, case law drawn from their own history, onboarding gates for rookies, retirement that hands insight to the next generation. You can't copy a decade — you have to live it.

Proof: every DNA version is a signed link in a chain — walk it offline from birth to today and verify each step.
